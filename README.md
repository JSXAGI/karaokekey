# KaraokeKey (PWA)

カラオケ好きなユーザーが、曲ごとの「自分に最適なキー」をメモしておくためのシンプルでスタイリッシュなWebアプリです。

## 特徴
- **オフライン対応**: PWA（Progressive Web App）として動作し、オフラインでも選曲リストの確認が可能です。
- **直感的な操作**: 曲名、アーティスト、キー（-4〜+4、原曲キー）を素早く登録できます。
- **データ永続化**: 入力データはブラウザの `localStorage` に保存されるため、アプリを閉じても消えません。
- **モバイルファースト**: ダークモードを採用したモダンなUIで、暗いカラオケルーム内でも眩しくありません。

## ファイル構成
- `index.html`: アプリ本体（構造、デザイン、ロジック）
- `manifest.json`: PWA設定ファイル（アイコン、テーマカラー、起動設定）
- `sw.js`: サービスワーカー（オフライン動作のためのキャッシュ制御）
- `README.md`: プロジェクト説明書
- `icon_192.png`: アプリ用アイコン（192px）
- `icon_512.png`: アプリ用アイコン（512px）

## セットアップ
1. 全ファイルをGitHubリポジトリにアップロードします。
2. GitHub Pagesなどの静的ホスティングサービスで公開します。
3. モバイルブラウザ（ChromeやSafari）でアクセスし、「ホーム画面に追加」を行うことでネイティブアプリのように利用可能になります。

## 技術スタック
- HTML5 / CSS3 (CSS Variables, Flexbox)
- JavaScript (Vanilla JS, Service Workers)
- LocalStorage API
